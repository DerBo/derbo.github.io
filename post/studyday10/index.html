数据结构与算法分析：最简单的排序算法之一，插入排序由N-1趟排序组成。对于p=1到N-1趟，插入排序保证从位置0到位置p上的元素为已排序状态。插入排序利用了这样的原理：位置0到位置p-1上的元素已经处于排过序的状态。

一 、代码

    // 插入排序
	    public void ChaRu(int[] arr) {
		    int j = 0;
		    for (int i = 1; i < arr.length; i++) {
                //保存当前位置元素的内容
			    int p = arr[i];
                //每一次都把前一个大于此元素的元素排在后面已经排好的队列当中
			    for (j = i; j > 0 && arr[i] < arr[j - 1]; j--) {
				   arr[j] = arr[j - 1];
				   arr[j - 1] = p;
			    }
		    }
	    }

原始数组   34    8    64    51    32    21    移动的位置

p=2趟后    8    34    64    51    32    21       0
p=3趟后    8    34    51    64    32    21       1
p=4趟后    8    32    34    51    64    21       3
p=5趟后    8    21    32    34    51    61       4

二 、 复杂度

  • 由于嵌套循环的每一个都花费N此迭代，因此插入排序的复杂度为O(N^2),而如果对一个已经预先排序的数列，它的运行时间就变为O(N)。 如果输入的数列几乎被排序，那么插入排序将执行的很快，不过插入排序和其他排序算法一样，对于平均的情形也是O(N^2)
