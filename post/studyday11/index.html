一种通过比较相距一定间隔的元素进行排序，每一趟比较所用的距离随着算法的进行而减小，直到只比较相邻元素的最后一趟排序位置，所以希尔排序有时也叫做缩减增量排序


        public void ShellSort(int[] arr) {
            // 3,2,9,3,6,5,8 length()7 n=7
            //定义增量的最大值N
		    int N = arr.length;
            //内部循环每运行一次 增量减少一半
		    for (int h = N / 2; h > 0; h /= 2) {
			    for (int i = h; i < N; i++) {
				    for (int j = i; j >= h && arr[j] < arr[j - h]; j -= h) {
                        //交换元素
					    arr[i] = arr[j];
					    arr[j] = arr[j - h];
					    arr[j - h] = arr[i];
				    }
			    }
		    }
	    }
  • 对于h，h+1…,N-1中的每一个位置i，把其上的元素放到i，i-h，i-2h，…中的正确位置上，通过仔细观察可以发现，没一趟h排序的作用就是对h个独立的子数组执行一次插入排序。
