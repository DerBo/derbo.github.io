一 、break

        经常使用在 switch 等各种循环语句中，作用是跳出当前循环并返回外部循环
   的起始位置开始第二次循环，如果循环为for循环则迭代表达式会自增

   ①在选择结构switch语句中

   ②在循环语句中

   ③离开使用场景的存在是没有意义的

    public static void main(String[] args){
        java.util.Scanner input = new java.util.Scanner(System.in);
        System.out.print("请输入一个正整数：");
        int num = input.nextInt();

        boolean flag = true;
        for (int i = 2; i <= Math.sqrt(num) ; i++) {
        if(num%i==0){
            flag = false;//这里即使break，i也会自增
            break;
        }
        }
        System.out.println(num + (flag?"是":"不是") + "素数");
    }

二 、continue

        经常使用在：结束本次循环，继续下一次的循环，如果main进程遇到continue则
    立即结束本次循环，同时迭代表达式也会自增

    //打印1-100之间的整数，跳过7的倍数和7结尾的数
    public static void main(String[] args){
    //打印1-100之间的整数，跳过7的倍数和7结尾的数
    for(int i=1; i<=100; i++){
        if(i%7==0 || i%10==7){
            continue;
            //break;
        }
        System.out.println(i);
    }
    }

三 、return

        经常使用在：
            ①结束循环：这点和break、continue不同，return会直接结束全部循环并结束进程
            ②用于返回某个值：格式 return 返回值；用于结束循环(如果有的话)并把返回值传递给
        上层调用程序

一 、嵌套循环

    嵌套循环：是指一个循环的循环体是另一个循环，比如for循环内还有一个for循环，那就是嵌套循环。总共的循环
次数=外循环次数，也可以是任意三种循环的互相嵌套

    嵌套循环格式：

        for(初始化语句①; 循环条件语句②; 迭代语句⑦) {
            for(初始化语句③; 循环条件语句④; 迭代语句⑥) {
                    循环体语句⑤;
            }
        }

        //break与双重循环
        //打印5行直角三角形
        public static void main(String[] args){
            for (int i = 0; i < 5; i++) {
                for (int j = 0; j < 5; j++) {
                    System.out.print("*");
                if(i==j){
                    break;
                }
                }
                    System.out.println();
            }
        }
